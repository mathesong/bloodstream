# Citation

If you use bloodstream in your research, please cite **kinfitr** for now. A dedicated bloodstream publication is forthcoming.

When describing your methods, please specify that "bloodstream was used for blood analysis, which is a blood processing pipeline built around kinfitr [REF]".

## kinfitr references

An introduction to the package:

> Matheson, G. J. (2019). *Kinfitr: Reproducible PET Pharmacokinetic Modelling in R*. bioRxiv: 755751. <https://doi.org/10.1101/755751>

A validation study compared against commercial software:

> Tjerkaski, J., Cervenka, S., Farde, L., & Matheson, G. J. (2020). *Kinfitr — an open source tool for reproducible PET modelling: Validation and evaluation of test-retest reliability*. EJNMMI Res 10, 77. <https://doi.org/10.1186/s13550-020-00664-8>
