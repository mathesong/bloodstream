# Container usage

For a quick introduction to running bloodstream with Docker or Apptainer, see the [Quick start](../quickstart.md). This section covers advanced container options, CLI reference, and HPC integration.

bloodstream provides Docker and Singularity/Apptainer container images that bundle all dependencies. Both use the same command-line arguments and produce identical results.

- **Docker** — best for desktops and cloud servers. See [Docker](docker.md).
- **Singularity/Apptainer** — best for HPC clusters and shared systems. See [Singularity/Apptainer](singularity.md).

```{toctree}
:maxdepth: 2

singularity
```
